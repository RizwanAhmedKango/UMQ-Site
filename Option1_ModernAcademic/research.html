Research Areas

01. Industrial IoT

Developing secure, decentralized communication protocols for mission-critical industrial environments. Focus on edge computing and real-time data synchronization at the network fringe.

Key project: "Resilient Edge Communication Frameworks for Smart Factories"

02. Applied AI

Integrating deep learning models for predictive maintenance and behavioral analysis in autonomous fleets. Leveraging multi-sensor data fusion to improve decision-making accuracy.

Key project: "Deep Learning for Autonomous Fleet Behavior Modeling"

03. Cyber-Physical Security

Investigation of lightweight cryptographic algorithms and zero-trust security architectures tailored for resource-constrained IoT devices.

Key project: "Post-Quantum Security for Wireless Sensor Networks"
