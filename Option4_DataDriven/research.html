Research Projects

Connected and Autonomous Systems

Project focus: integration of UWB and accelerometry for high-precision tracking.

Leading the development of a hybrid localization system that combines Ultra-Wideband (UWB) and inertial sensing (accelerometers) for monitoring livestock and autonomous vehicle behavior. This project addresses the challenge of precise tracking in varied outdoor and industrial environments.

Industrial IoT for Smart Manufacturing

Project focus: edge-computing protocols and security.

Investigation of secure communication protocols for large-scale Industrial IoT networks. This research develops lightweight encryption and authentication methods suitable for resource-constrained edge devices in smart factories.

Applied AI in AgTech

Project focus: predictive analytics and sensor integration.

Utilizing machine learning models to analyze multi-spectral sensor data from agricultural fields. The project seeks to build predictive models for crop health and yield estimation, directly impacting sustainable farming practices.

Information Security in Wireless Sensor Networks

Project focus: threat detection and data integrity.

Researching robust mechanisms for defending against cyber-physical attacks on wireless sensor networks. We develop anomaly detection algorithms that differentiate between sensor malfunctions and malicious interference.
